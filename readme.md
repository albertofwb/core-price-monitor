## Description

> get below information from https://stake.coredao.org and push with telegram bot

- total BTC staked
- total CORE stated
- validator staked
- validator reward 

## environment

 
- OS Ubuntu 22.04
- Python 3.10.0

## How to run


**fill in your telegram token and chat_id to the .env file**

```bash
pip install -r requirements.txt
cd src
python3 push_core_price.py
```
